# 🧬 AADNA Local CMS

Быстрая, легкая и полностью автономная локальная админ-панель для управления ДНК-результатами сайта **[aadna.ru](https://aadna.ru)**. 

Этот проект разработан в премиальной дизайн-системе **Apsny Production Inc. (API)** и заменяет собой облачную Pages CMS, работая напрямую с файловой системой вашего компьютера. Это полностью решает проблему сетевых задержек и сбоев API GitHub (ошибки 503).

---

## ✨ Ключевые возможности

* **Мгновенная работа**: сохранение файлов на диск происходит за 0 миллисекунд без сетевых запросов.
* **Автоматизация бизнес-логики**:
  * **Авто-транслитерация**: автоматическое превращение русских заголовков в безопасные URL-слаги.
  * **Синхронизация таксономий**: автоматическое наполнение полей фильтрации (гаплогруппы, этносы, селения, лаборатории) на основе введенных данных.
  * **Редиректы (Aliases)**: автоматическое сохранение старых URL при переименовании поста.
  * **SNP-цепочки**: скачивание цепочек снипов по API с `snp.apsny.dev` напрямую в `data/paths/{snp}.json`.
  * **OG-превью**: генерация картинок 1200x630 для соцсетей (с флагом этноса, логотипом лаборатории, QR-кодом и фамилией) с помощью библиотеки `sharp`.
* **Умный загрузчик медиа**:
  * Поддержка **Drag & Drop** и вставки изображений из буфера обмена (**Ctrl+V**) прямо в текстовые поля.
  * Автоматическое переименование и перемещение картинок в именную структурированную папку рода (`static/media/results/{slug}/{slug}_N.png`).
* **Интегрированный предпросмотр**: сохранение поста в один клик и автоматическое открытие вкладки на локальном сервере Zola (`http://localhost:1111`) для моментального просмотра результата.
* **Git-интеграция**: коммит и публикация изменений на GitHub прямо из админки в один клик.
* **Умный запуск (Auto-Update)**: при запуске админки скрипт проверяет наличие свежих коммитов на GitHub для CMS и сайта и автоматически обновляет их (`git pull --rebase`), если нет локальных конфликтов.

## 🚀 Установка для пользователей (Установщик в 1 клик)

Если вы не являетесь разработчиком, самый простой способ начать работу — использовать наш автоматический установщик.

### Инструкция для Windows:
1. Скачайте два файла: **`install.bat`** и **`install.ps1`** из этого репозитория и положите их в одну папку (например, на Рабочий стол).
2. Дважды кликните по файлу **`install.bat`**.
3. Нажмите **Enter**, чтобы согласиться на папку установки по умолчанию (`C:\aadna-workspace`).
4. Скрипт проверит и при необходимости сам установит Git, Node.js и GitHub CLI.
5. На этапе авторизации GitHub в консоли появится 8-значный код, и **автоматически откроется браузер**. Введите этот код на сайте GitHub и нажмите **Authorize** (у вашего аккаунта должен быть доступ к репозиторию сайта `aadna`).
6. По завершении установки на вашем Рабочем столе появится ярлык **`AADNA CMS`**. Дважды кликните по нему для запуска!

### Инструкция для macOS / Linux:
1. Скачайте файл **`install.sh`**.
2. Запустите его в терминале:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
3. Следуйте простым инструкциям на экране.

## 🛠 Ручная установка (для разработчиков)

Если у вас уже настроено рабочее окружение и клонирован репозиторий сайта `aadna`:

1. Клонируйте этот репозиторий в ту же общую папку, где лежит проект сайта (так, чтобы они были соседними папками):
   ```
   workspace/
   ├── aadna/
   └── aadna-local-cms/
   ```
2. Перейдите в папку CMS и установите зависимости:
   ```bash
   cd aadna-local-cms
   npm install
   ```
3. Запустите сервер:
   ```bash
   npm start
   # Или запустите run.sh / run.bat напрямую
   ```
4. Откройте в браузере: **`http://localhost:4400`**

## 📂 Структура проекта

```
aadna-local-cms/
├── server.js          # Express-сервер, обрабатывающий API и загрузку файлов
├── lib/               # Логика автоматизаций (модули Node.js)
│   ├── git.js         # Работа с Git (status, commit, push, pull)
│   ├── media.js       # Загрузка и релокация изображений
│   ├── normalize.js   # Нормализация YAML-frontmatter и таксономий
│   ├── preview.js     # Генератор OG-превью изображений (sharp + SVG)
│   ├── slugify.js     # Транслитерация кириллицы в латиницу
│   └── snp.js         # API клиент для snp.apsny.dev
├── public/            # Фронтенд (SPA-приложение)
│   ├── index.html     # Структура интерфейса
│   ├── styles.css     # Premium UI styling (тёмная тема, glassmorphism)
│   ├── app.js         # Логика SPA, рендеринг формы и отправка данных
│   └── editor.js      # Помощник Markdown-редактора (Ctrl+V, D&D)
├── install.bat        # Windows-лаунчер установщика
├── install.ps1        # PowerShell скрипт установки для Windows
├── install.sh         # Bash скрипт установки для macOS/Linux
├── run.bat            # Windows запускатор CMS с автообновлением
└── run.sh             # macOS/Linux запускатор CMS с автообновлением
```
